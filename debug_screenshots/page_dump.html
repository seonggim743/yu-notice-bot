파일 추가

문서를 서버에 보내지 않고, 사용자의 장치에서만 동작합니다.

사용자 접속 환경에 따라 시간이 소요될 수 있습니다.

사용법

  1. 디바이스에서 파일을 선택 하거나, 드래그 앤 드롭 하세요.(최대 5개, 30MB까지 지원)
  2. 파일 리스트에 선택한 파일이 정상적으로 보여지는지 확인 하세요(암호 문서의 자물쇠 버튼을 눌러 암호 해제).
  3. 기능 버튼을 눌러 완료 되면, 다운로드 버튼을 눌러 결과 파일을 받으세요.
  4. 결과 파일을 Google Drive에 업로드할 수 있습니다.

보안

사용자의 파일을 서버에 업로드 하지 않습니다.

tools의 기능 동작 중에도 장치 내부에서만 문서가 사용 됩니다.

특징

문서의 페이지들을 큰 크기-고품질, 중간 크기-일반품질, 작은 크기-저품질로 변환해 줍니다.

zip 파일안의 문서들도 JPG로 변환해 줍니다.

변환된 이미지는 페이지별 이름이 붙어서 하나의 압축 파일안에 존재합니다.

기능이 성공하면 다운로드 버튼이 활성화되고 실패시 에러 팝업이 뜹니다.
